# 命名规范
***
## 匈牙利命名法
### 1、类型命名
|  类型         | 前缀 |类型        |实例|
|  ----         | ----|----         | ----          |
|  整数         | i  |Integer       | iItemCount    |
|  数组         | a  |Array         | aItems        |
|  函数         | fn |Function      | fnHandler     |
|  对象         | o  |Object        | oDiv1         |
|  浮点数       | f  |Float         | fPrice        |
|  字符串       | s  |String        | sUserName     |
|  布尔值       | b  |Boolean       | bIsComplete   |
|  变体变量     | v  |Variant       | vAnvthing     |
|  正则表达式   | re |RegExp        | reEmailCheck  |
### 2、驼峰式命名
> 类型名小写或第一个单词首字母小写
> 
> 后面单词的首字母大写
